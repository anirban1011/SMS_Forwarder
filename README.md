# SMS Forwarder

A high-performance Android automation utility designed to automatically forward incoming SMS messages to **webhooks, email addresses, or other phone numbers**.

## 📱 Installation Guide

1. Open your device **Settings** and enable **Install unknown apps** for the browser or file manager you will use to install the APK.

2. Temporarily **pause Google Play Protect** from the Google Play Store if it prevents the APK from being installed.

3. Download and install the **SMS Forwarder APK** using the link provided above.

4. Open the app and **disable battery optimization** for SMS Forwarder to help ensure reliable background operation.

5. Open **App Info**, tap the **three-dot menu** in the top-right corner, and select **Allow restricted settings**.

6. Open **Permissions** and grant all three required permissions:
   - **Notifications**
   - **Phone**
   - **SMS**

7. Setup is complete! Open **SMS Forwarder** and create your first **Forwarding Rule**.

## ✨ Features

- 📱 Forward incoming SMS messages to another phone number
- 📧 Forward SMS messages to email
- 🌐 Send SMS data to custom webhooks
- ⚙️ Rule-based SMS forwarding
- 🔄 Background operation
- ⚡ Designed for reliable and high-performance SMS automation

## ⚠️ Important Note

Some Android manufacturers use aggressive battery management systems that may prevent apps from running reliably in the background.

For uninterrupted SMS forwarding, make sure to:

- Disable **Battery Optimization** for SMS Forwarder.
- Allow the app to **run in the background**.
- Grant all required **SMS, Phone, and Notification permissions**.
